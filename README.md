# react-app-boilerplate

```
$ git clone git@github.com:crazytoucan/react-app-boilerplate.git
$ yarn
$ yarn start
# navigate to http://localhost:8080
```

## Features

- TypeScript
- LESS
- React hot loading
- LESS hot loading

Notably, this boilerplate does **not** use or support:

- Testing (yet)
- Blueprint (yet)
- Building distributable assets (yet)
- Babel
